# 小桂助手 (XiaoGui-Assistant)

## 项目简介
这是二人合作开发的本地 AI 助手后端。
核心基于 DeepSeek 模型，旨在整合日常高频需求：时间查询、基础计算、文档解析、联网搜索及图片识别。

目前项目核心功能链路已打通。，当前版本可供实战项目基本需求。

## 主要功能
- **智能对话**：基于 DeepSeek 模型的多轮上下文问答。
- **文档解析**：支持上传 PDF、Word 或图片，自动提取并分析内容。
- **联网搜索**：集成实时搜索工具，可获取最新网络信息（需配置接口）。
- **实用工具**：内置本地时间查询与基础数学计算器。
- **图像识别**：支持对上传图片进行场景理解与文字提取。
